# 多图像选择器工具使用说明

## 功能概述

多图像选择器工具是一个增强版的图像源工具，支持：
- **加载多张图片**：一次选择多张图片进行处理
- **图片切换**：提供上一张/下一张按钮进行切换
- **自动运行**：切换图片后自动执行整个流程
- **循环模式**：支持循环浏览（到最后一张后回到第一张）

## 使用方法

### 1. 添加工具

1. 在工具库的"图像源"分类中找到"多图像选择器"
2. 拖拽到算法编辑器中

### 2. 加载图片

1. 双击工具或点击属性面板中的"选择图片..."按钮
2. 在文件对话框中选择多张图片（支持 Ctrl/Shift 多选）
3. 点击"打开"加载图片

也可以点击"+"按钮添加更多图片到现有列表。

### 3. 切换图片

工具提供多种切换方式：

- **上一张/下一张按钮**：点击"◀ 上一张"或"下一张 ▶"按钮
- **快速跳转**：输入图片序号点击"跳转"按钮
- **自动循环**：开启循环模式后，到最后一张会自动回到第一张

### 4. 自动运行

勾选"切换图片后自动运行流程"选项后：
- 每次切换图片都会自动触发整个流程的执行
- 无需手动点击运行按钮

### 5. 属性说明

| 属性 | 说明 |
|------|------|
| 图像文件列表 | 显示已加载的所有图片路径 |
| 当前图像索引 | 当前选中的图片序号 |
| 自动运行 | 开启后切换图片自动执行流程 |
| 循环模式 | 开启后到最后一张会回到第一张 |

## 使用示例

假设你需要处理一个包含100张产品图片的文件夹：

1. 添加"多图像选择器"工具到流程
2. 加载所有100张图片
3. 添加需要的处理工具（如图像滤波、模板匹配等）
4. 开启"自动运行"选项
5. 点击"下一张"按钮，工具会自动处理每张图片

## 注意事项

1. **图片路径**：图片加载后会保存相对路径或绝对路径，移动图片文件可能导致路径失效
2. **内存占用**：加载大量图片不会占用内存，只有当前选中的图片会被读入内存
3. **自动运行**：确保流程配置正确后再开启自动运行，避免重复错误
4. **结果保存**：建议配合"图像保存"工具使用，自动保存每张图片的处理结果

## 技术细节

- **工具类型**：`ImageSource`
- **输出端口**：`OutputImage`
- **附加输出**：`CurrentIndex`, `TotalImages`, `CurrentPath`, `FileName`
- **中文参数名**：图像文件列表、当前图像索引、自动运行、循环模式
